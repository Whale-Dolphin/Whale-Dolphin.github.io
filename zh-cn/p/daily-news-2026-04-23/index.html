🔥 头条

SpaceX 获得以 $600 亿收购 Cursor 的选项

SpaceX 宣布获得一项独特协议：今年晚些时候可以 $600 亿收购 AI 代码生成创业公司 Cursor，或者支付 $100 亿用于双方合作项目。Cursor 是当前最热门的 AI 编程编辑器之一，这笔交易如果完成，将是 AI 工具领域迄今为止最大的收购案。SpaceX 进军 AI 编程工具领域的意图引发行业广泛关注。

🔗 Reuters · CNBC

Google 发布第八代 TPU：为 Agent 时代设计的双芯片架构

Google 在 Cloud Next 大会上推出第八代 TPU，首次采用双芯片设计：TPU 8t 专为训练优化，可在单一内存池中运行最复杂模型；TPU 8i 专为推理优化。Google 将其定位为"Agent 时代"的基础设施，瞄准了 AI Agent 大规模部署对算力的新需求。

🔗 Google Blog · CNBC

Anthropic 调查 Claude Mythos 未授权访问事件

Anthropic 正在调查其独家网络安全 AI 模型 Claude Mythos 遭未授权访问的指控。有报道称一小群人通过第三方供应商环境获得了 Mythos 的访问权限。Mythos 是 Anthropic 于 4 月 7 日发布的网络安全专用模型，因能力过强而未对公众开放。这起事件凸显了高性能 AI 模型在安全管控上面临的挑战。

🔗 TechCrunch · BBC · Reuters

🛠 工具与框架

Zed 推出 Parallel Agents — 并行 AI 编程助手

Zed 编辑器正式推出 Parallel Agents 功能，允许开发者同时运行多个 AI Agent 执行不同任务。同时新版还加入了 Claude Opus 4.7 支持（BYOK）、块注释切换、滚轮缩放字体等特性。这标志着 AI 编程编辑器正从"单线程对话"向"多 Agent 并行"演进。

🔗 Zed Blog · HN 讨论

SGLang — 高性能 LLM 与多模态模型推理框架

SGLang 是一个面向大语言模型和多模态模型的高性能服务框架，目前在 GitHub 上已获得 26,276 颗星。它提供了灵活的推理引擎和高效的批处理能力，是 vLLM 的有力竞争者之一。

🔗 GitHub ⭐ 26.3K

Context7 — 面向 LLM 的实时代码文档平台

Context7 为 AI 代码编辑器和 LLM 提供最新的代码文档，解决了 LLM 使用过时 API 文档的问题。目前在 GitHub 上已有 53,467 颗星，是本周最热门的 AI 开发工具之一。

🔗 GitHub ⭐ 53.5K

Headroom — LLM 应用的上下文优化层

Headroom 定位为 LLM 应用的"Context Optimization Layer"，帮助开发者更高效地管理和优化传递给 LLM 的上下文信息，减少 token 浪费和幻觉产生。已获得 1,489 颗星。

🔗 GitHub ⭐ 1.5K

ACI.dev — 600+ 工具的 AI Agent 调用平台

ACI.dev 是一个开源的 AI 工具调用平台，将 600+ 工具接入任意 AI Agent IDE 或自定义 Agent，支持直接函数调用或统一 MCP 服务器。它提出了"VibeOps"的概念——让 AI Agent 的工具调用变得像"直觉"一样自然。

🔗 GitHub ⭐ 4.8K

SpiceAI v2.0-rc.3 — 便携式 SQL + LLM 推理引擎

SpiceAI 是一个用 Rust 编写的便携式加速 SQL 查询、搜索和 LLM 推理引擎，专为数据驱动的 AI 应用和 Agent 设计。最新版 v2.0-rc.3 修复了 GitHub 连接器问题并升级了 Rust 工具链。

🔗 GitHub ⭐ 2.9K · Release Notes

Intel Arc Pro B70 — 32GB VRAM 的本地推理新选择

Intel Arc Pro B70 搭载 32GB VRAM，定位为本地 AI 推理显卡而非游戏卡。目前社区已有 llama.cpp 优化补丁（2-7x 提速）和 vLLM 支持。不过软件生态成熟度仍是关键挑战。

🔗 BuzzRAG 评测 · 优化补丁

Linux 内核因 AI 生成安全报告移除业余无线电协议栈

Linux 内核开发者宣布移除业余无线电（AX.25、NET/ROM、ROSE）协议实现及所有相关驱动。原因令人深思：这些协议长期是 syzbot bug 报告的"磁铁"，而 AI 生成的大量安全报告让维护者不堪重负，无人愿意帮忙处理，最终只能将代码移出内核树。这是 AI 自动化安全审计对开源社区维护压力的一个缩影。

🔗 LWN.net · HN 讨论

Accurate, Interdisciplinary and Transparent Structure-property Understanding with Deep Native Structural Reasoning

SciReasoner 把蛋白质、小分子和晶体结构转成可检查的结构 token，让科学结构-性质推理同时提升准确率和可解释性。

  • 作者: Chen Tang et al.
  • 来源: HuggingFace Trending (77 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 论文提出 SciReasoner，用统一的 structure-aware vocabulary 表示坐标、拓扑和周期连接，让模型能把结构证据作为推理单元使用。它在 86 个 benchmark 中有 67 个达到 SOTA，并在蛋白质 GO 预测、单步逆合成、材料相/带隙区分上给出可解释推理链，对科学 AI 和材料/生物/化学建模很有参考价值。
  • 相关技术: structure-aware vocabulary, native structural reasoning, Gene Ontology prediction, retrosynthesis, crystal structure modeling
  • 代码/权重: 未提及
📄 Abstract 中文翻译

结构-性质关系是生物学、化学和材料科学的基础，功能、反应性和物理响应都来自空间、化学和周期性组织。要从机理上解释这些关系，需要通过科学原理和物理约束来解读结构证据，范围从立体化学和成键，到对称性、能量学和周期性有序结构。然而，将人工智能用于这一过程面临表示与推理的共同挑战：模型既要保留领域原生的结构信息，又要说明具体证据如何在这些约束下支持预测。本文提出 SciReasoner，这是一个面向蛋白质、小分子和无机晶体的原生结构推理多模态科学基础模型。SciReasoner 将坐标、拓扑和周期性连接离散化为统一的结构感知词表，并在推理过程中把结构 token 作为可寻址的证据单元。在同源性控制的 Gene Ontology (基因本体, GO) 预测中，SciReasoner 提升了低同源和孤儿样蛋白的 Cellular Component 注释表现，将 Fmax 从 0.42 提高到 0.55。在化学任务中，它将单步逆合成准确率从 0.63 提高到 0.72，同时生成片段级断键和前体验证轨迹。在材料科学中，它的表征能够区分元素相和化合物相，并解析高带隙与低带隙区域。在 86 个 benchmark 中，SciReasoner 在 67 个任务上达到最先进性能。双盲专家评估显示，其推理轨迹在 98% 的案例中被认为优于或至少可比于前沿大语言模型。通过让结构成为在科学约束下可检查的推理基底，SciReasoner 将准确预测与可解释科学推断连接起来。

UniClawBench: A Universal Benchmark for Proactive Agents on Real-World Tasks

UniClawBench 用 400 个双语真实任务和 live Docker 环境评估主动型 Agent，比静态单轮 benchmark 更接近实际使用。

  • 作者: Zhekai Chen et al.
  • 来源: HuggingFace Trending (22 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 论文把 Agent 评测从场景分类改成能力分类，覆盖技能使用、探索、长上下文推理、多模态理解和跨平台协作。它用执行器 Agent、隐藏监督 Agent 和用户 Agent 组成闭环评估，适合分析 base model 与 agent framework 各自造成的失败原因。
  • 相关技术: proactive agents, live Docker evaluation, step-by-step checkpoints, closed-loop evaluation, cross-platform coordination
  • 代码/权重: 已开源 ✅
📄 Abstract 中文翻译

大语言模型和多模态大语言模型的快速发展，加速了主动型 Agent 的出现，这类 Agent 能够操作日常工具，并在真实环境中辅助用户。然而，现有 benchmark 难以有效评估这类 Agent，因为它们通常依赖沙盒环境和单轮评估范式。此外，它们基于场景的任务分类会在同一任务类别中混合多种模型能力，使得研究者难以识别 Agent 失败的根本原因。为解决这些限制，本文提出 UniClawBench，这是首个面向动态真实环境、以能力为驱动的主动型 Agent benchmark。UniClawBench 围绕五种基础模型能力构建：Skill Usage (技能使用)、Exploration (探索)、Long-Context Reasoning (长上下文推理)、Multimodal Understanding (多模态理解) 和 Cross-Platform Coordination (跨平台协作)。基于这些能力，作者设计了 400 个双语真实世界任务。不同于依赖静态预录答案的以往 benchmark，该 benchmark 在 live Docker 容器中评估 Agent，并使用细粒度的逐步完成检查点。此外，作者设计了一个闭环评估策略，由执行器 Agent、隐藏监督 Agent 和用户 Agent 组成，用于模拟真实的多轮人类反馈，同时不泄露评分标准。为区分基础模型能力和框架级设计选择的影响，作者在多个 Agent framework 下评估了最先进模型。通过对模型和框架的全面比较，论文展示了基础模型能力与 Agent framework 设计如何共同影响真实环境中的表现。为了促进后续研究，作者公开了 benchmark 和代码：https://github.com/HKU-MMLab/UniClawBench。

Jet-Long: Efficient Long-Context Extension with Dynamic Bifocal RoPE

Jet-Long 用动态双焦 RoPE 在不微调的情况下扩展长上下文，同时尽量保住短上下文质量和推理吞吐。

  • 作者: Haozhan Tang et al.
  • 来源: HuggingFace Trending (7 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 方法把本地 RoPE-faithful window 和随序列长度动态缩放的 long-range window 结合起来，短输入时恢复 base model，长输入时稳定外推。它通过 inclusion-exclusion attention merge 和 on-the-fly RoPE correction rotation 降低推理成本，在 H100 上长上下文 prefill 最高达到 FA2 的 1.39 倍吞吐。
  • 相关技术: Dynamic Bifocal RoPE, zero-shot context extension, inclusion-exclusion attention, CuTe kernel, long-context prefill
  • 代码/权重: 未提及
📄 Abstract 中文翻译

现代 LLM 越来越多地部署在长上下文应用中，例如 retrieval-augmented generation (检索增强生成, RAG)、仓库级代码任务和 agentic workflows。这些场景中累积的推理与工具轨迹经常让输入长度超过预训练窗口一个数量级，使零样本上下文扩展成为开放权重 checkpoint 的主要部署路径。多数现有零样本方法会预先固定一个缩放因子，因此激进的缩放会损害短上下文保真度，而保守的缩放又会在长上下文中失效。本文提出 Jet-Long，这是一种无需调参的零样本方法，将一个忠实保持本地 RoPE 的窗口与一个长程窗口配对，后者的缩放因子会根据当前序列长度动态调整，从而在短输入时精确恢复基础模型，在长输入时干净外推。通过 inclusion-exclusion attention merge (容斥式注意力合并) 和在线 RoPE 校正旋转，双焦结构在推理中几乎没有额外成本；融合到单个 CuTe kernel 后，H100 上的长上下文 prefill 吞吐最高达到 FA2 的 1.39 倍，接近 Hopper-only FA4，单 batch 生成在任意长度下的额外开销不超过 4%。在 Qwen3-1.7B/4B/8B 上扩展到 128K context 时，Jet-Long 在 RULER 上分别比最强 baseline 高出 4.79/2.18/2.03 个百分点，在 HELMET-RAG 上取得最佳总体准确率，并获得最低的 PG-19 perplexity。Jet-Long 还能泛化到 Jet-Nemotron 等 hybrid attention architecture，在无需重新训练的情况下进一步改善长上下文能力，并且对超参数具有鲁棒性，便于部署。

Single-Rollout Asynchronous Optimization for Agentic Reinforcement Learning

SAO 针对长程 Agent RL 的异步训练稳定性问题，把 GRPO 式 group sampling 改成 single-rollout sampling，并加入双侧 token-level clipping。

  • 作者: Zhenyu Hou et al.
  • 来源: HuggingFace Trending (10 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 论文聚焦异步 RL 在 agentic coding/reasoning 中的 off-policy 和稳定性问题，用每个 prompt 一个 rollout 的训练方式提升泛化并减少组采样不匹配。SAO 能稳定训练 1000 步，在 SWE-Bench Verified、BeyondAIME、IMOAnswerBench 等任务上超过 GRPO 及其变体，并已用于开放 GLM-5.2 训练流水线。
  • 相关技术: asynchronous RL, single-rollout sampling, token-level clipping, value-model training, agentic coding benchmarks
  • 代码/权重: 未提及
📄 Abstract 中文翻译

Reinforcement Learning (强化学习, RL) 对大语言模型后训练变得越来越重要。以往用于 LLM 的 RL pipeline 大多是同步且 batch-interleaved 的，这对长程 Agent 任务来说效率较低。近期，异步 RL 作为一种更高效的替代方案出现，它会在 rollout 返回时更新模型。然而，现有异步 RL 系统通常强调吞吐，而对训练稳定性和任务有效性探索不足。例如，广泛使用的 GRPO 框架中的 group-wise sampling 并不天然适配异步 Agent 训练。本文提出 Single-rollout Asynchronous Optimization (单 rollout 异步优化, SAO)，用于解决异步 RL 中的稳定性和 off-policy 挑战。为减少 off-policy 影响并改善泛化，作者将 group-wise sampling 替换为 single-rollout sampling，即每个 prompt 只使用一个 rollout。作者进一步用实用的 value model 训练设计增强这一单 rollout 策略。为了提升优化稳定性，论文引入严格的双侧 token-level clipping 策略。SAO 能够稳定训练 1000 步，并在 SWE-Bench Verified、BeyondAIME 和 IMOAnswerBench 等 Agent coding 与 reasoning benchmark 上持续超过 GRPO 及其变体。论文还展示了 single-rollout RL 在模拟在线学习设置中特别有效，在这种设置下模型必须适应不断变化的演化环境。最终，SAO 已成功部署到开放 GLM-5.2 模型 (750B-A40B) 的 Agent RL 训练 pipeline 中。

PS4: Proxy-Supervised Joint Training for Real Target Speaker Extraction

PS4 为真实对话混合语音中的目标说话人提取构造代理监督训练方案，在 REAL-T challenge 中拿到整体第 2。

  • 作者: Wanyi Ning et al.
  • 来源: HuggingFace Trending (0 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 论文构造了 71,771 条中英双语训练样本，每条包含重叠语音、说话人 enrollment audio、转录和帧级 VAD 标签。训练上同时使用 ASR cross-entropy、speaker similarity、VAD 和 perceptual audio quality 四种可微目标，只微调 BSRNN separator，对真实会议语音分离/提取很实用。
  • 相关技术: target speaker extraction, proxy-supervised training, BSRNN separator, voice activity detection, REAL-T challenge
  • 代码/权重: 未提及
📄 Abstract 中文翻译

在真实对话混合语音中训练 Target Speaker Extraction (目标说话人提取, TSE) 模型仍然很困难，因为缺少大规模训练语料和用于监督的干净目标语音。本文提出 PS4，这是一个用于真实对话混合语音中 TSE 的 proxy-supervised training (代理监督训练) 框架，包含两个主要贡献。第一，作者从四个公开数据集中构建了一个包含 71,771 条训练样本的大规模语料，覆盖中文和英文场景。每条样本都包含一段重叠语音混合、每个说话人的 enrollment audio (注册音频)、ground-truth transcript (真实转录) 和帧级 voice activity labels (语音活动标签)。第二，作者提出一种代理监督联合训练策略，用四个互补的可微目标微调基于 BSRNN 的 TSE 模型：ASR cross-entropy (自动语音识别交叉熵)、speaker similarity (说话人相似度)、frame-level voice activity detection (帧级语音活动检测) 和 perceptual audio quality (感知音频质量)。从公开可用的预训练 checkpoint 出发，微调过程中只更新 BSRNN separator。在 REAL-T challenge leaderboard 上，PS4 总体排名第 2，并在所有提交系统中取得最佳的说话人相似度和 timing F1。

LongE2V: Long-Horizon Event-based Video Reconstruction, Prediction, and Frame Interpolation with Video Diffusion Models

LongE2V 用预训练视频扩散先验处理事件流视频重建、预测和插帧，重点解决长序列时序漂移。

  • 作者: Cheng-De Fan et al.
  • 来源: HuggingFace Trending (19 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 方法把 event-based vision 和 video diffusion prior 结合，通过 Autoregressive Unrolling 与 Adaptive Context Switching 提升超长序列稳定性。对事件相机视频重建和插帧任务有直接参考价值，尤其是稀疏事件流下的感知质量和时序一致性。
  • 相关技术: event-based video reconstruction, video diffusion priors, autoregressive unrolling, adaptive context switching, frame interpolation
  • 代码/权重: 未提及
📄 Abstract 中文翻译

从稀疏事件流中恢复高质量视频是一项具有挑战性的任务。回归方法通常会让纹理变得模糊，而现有生成式模型又难以保持长期稳定性。本文提出 LongE2V，这是一种利用预训练视频扩散先验，同时处理 event-based video reconstruction (基于事件的视频重建)、prediction (预测) 和 frame interpolation (帧插值) 的新方法。通过微调基础视频模型，该方法实现了高数据效率和更好的感知质量。作者引入 Autoregressive Unrolling (自回归展开) 和 Adaptive Context Switching (自适应上下文切换) 来缓解极长序列中的时间漂移。作者还提出 Reencoding Alignment with Cross Residual Correction (带交叉残差校正的重编码对齐)，以确保帧插值中的精确双向一致性。此外，Event Voxel Density Augmentation (事件体素密度增强) 保证了模型在不同传感器分辨率下的鲁棒性。在真实世界 benchmark 上的大量实验表明，LongE2V 在三项任务上均超过现有最先进方法，表现出优秀的时间一致性和零样本泛化能力。项目页面：https://cdfan0627.github.io/LongE2V-page/。

Who Broke the System? Failure Localization in LLM-Based Multi-Agent Systems

AgentLocate 试图定位多 Agent 系统失败时是哪一个 Agent、哪一步让轨迹不可逆地跑偏。

  • 作者: Yufei Xia et al.
  • 来源: HuggingFace Trending (0 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 论文把多 Agent failure localization 拆成责任 Agent 识别和最早决定性错误步骤定位。AgentLocate 用 LLM judge、多视角独立验证和 confidence-aware aggregation 组合判断，再用反馈轻量微调 judge，对复杂 agent workflow 的日志诊断很有价值。
  • 相关技术: multi-agent systems, failure localization, LLM judge, multi-perspective verification, confidence-aware aggregation
  • 代码/权重: 未提及
📄 Abstract 中文翻译

基于 Large Language Model (大语言模型, LLM) 的 multi-agent systems (多智能体系统) 能通过协同推理和行动解决复杂问题，但其分布式结构也为诊断系统级失败带来了新挑战。当一次执行失败时，由于长程交互和紧密耦合的 Agent 行为，很难识别哪个 Agent 应负责，以及轨迹在何处第一次变得不可逆地偏离目标。本文研究 LLM-based multi-agent systems 中的 failure localization (失败定位) 问题，并提出 AgentLocate，一个同时将失败归因到具体 Agent 和最早决定性步骤的框架。AgentLocate 将基于 LLM 的 judging mechanism 与独立评估器的多视角验证结合，并用 confidence-aware strategy (置信度感知策略) 聚合评估结果。得到的反馈进一步用于通过轻量微调来适配 judge，从而提升归因质量。作者在两个互补 benchmark 上评估 AgentLocate，覆盖多样任务、Agent 配置和轨迹长度。实验结果表明，AgentLocate 在识别责任 Agent 和失败步骤方面持续优于现有 failure localization 方法，同时在 token 使用量和运行时间上保持高效。

Scaling Mixture-of-Experts Video Pretraining for Embodied Intelligence

LingBot-Video 是面向具身智能的视频预训练范式，用 MoE DiT 和机器人数据 profiling 缓解通用视频生成模型的物理世界错配。

  • 作者: Shuailei Ma et al.
  • 来源: HuggingFace Trending (42 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 论文从架构、数据和训练三方面把视频生成模型改造成具身智能基础模型：MoE 提升容量/效率权衡，数据 profiling 加入机器人操作、导航和 egocentric 视频，多维 reward 强化物理合理性和任务完成度。对 VLA 和 robotics world model 的预训练路线很有参考价值。
  • 相关技术: Mixture-of-Experts, DiT video pretraining, embodied intelligence, robot-oriented footage, multi-dimensional reward
  • 代码/权重: 已开源 ✅
📄 Abstract 中文翻译

UP: Unbounded Positive Asymmetric Optimization for Breaking the Exploration-Stability Dilemma

UP 重新设计 RL objective，让正优势样本保持无界稳定梯度，负优势样本继续用 clipping 防止崩坏。

  • 作者: Chongyu Fan et al.
  • 来源: HuggingFace Trending (2 upvotes)
  • 链接: arXiv | PDF
  • 关键贡献: 论文指出保守 clipping 会过早截断正确但低置信推理路径的更新预算，限制探索。UP 用 stop-gradient 将策略锚定到当前状态，对正优势释放 unclipped gradient，对负优势保持标准 clipping，并可接入 GRPO、DAPO、GSPO 等 token/sequence 级框架。
  • 相关技术: Unbounded Positive Asymmetric Optimization, importance sampling, probability capacity, GRPO, DAPO
  • 代码/权重: 未提及
📄 Abstract 中文翻译

Reinforcement Learning (强化学习, RL) 已成为增强大语言模型复杂推理能力的标准范式。为实现样本效率，现代 RL 框架依赖 importance sampling (重要性采样, IS)。然而，这些算法存在 exploration-stability dilemma (探索-稳定性困境)。纯 IS 往往导致灾难性训练不稳定，而用于缓解不稳定性的标准 clipping 机制又会严格限制策略更新预算。通过形式化 Probability Capacity (概率容量, Cap) 概念，作者揭示保守 clipping 会结构性地抑制探索，因为它过早截断了正确但低置信推理路径的更新预算。为摆脱这些限制，本文提出 Unbounded Positive Asymmetric Optimization (无界正向非对称优化, UP)，一个通用且可插拔的目标函数。UP 通过 stop-gradient operator (停止梯度算子) 将策略锚定到当前状态，从理论上重构优化过程。这种非对称设计为 positive advantages (正优势) 释放无 clipping 且稳定的梯度以最大化探索，同时为 negative advantages (负优势) 保留标准 clipping 防护来防止训练不稳定。此外，该公式可以自然扩展到不同优化粒度，包括 token-level 的 GRPO、DAPO 和 sequence-level 的 GSPO。大量实验表明，UP 能增强探索容量，并在多种 RL 算法、模型架构和训练模态上取得更好的推理准确率，覆盖 DAPO、GSPO、GRPO，Dense、MoE、vision-language，以及语言和多模态训练，验证了 UP 是一个真正通用的 RL-based training 可插拔增强方法。

📄 Abstract 中文翻译

Time Series Classification (时间序列分类, TSC) 主要由两阶段范式主导：先训练一个 feature encoder，可以是在目标数据集上从零训练，也可以在大规模语料上预训练，然后在其上拟合任务特定分类器。虽然这种方法有效，但它将表示学习与分类目标分离优化，需要针对每个数据集训练，并且阻止模型在推理时利用标签信息。本文提出 TimEE，一个 4.5M 参数的基础模型，通过 in-context learning (上下文学习, ICL) 进行端到端 TSC。给定一个 labeled support set (带标签支持集) 和一个 query time series，TimEE 在单次前向传播中直接输出预测类别分布，不需要任何 per-dataset training (针对每个数据集的训练)。遵循 prior-data fitted network (先验数据拟合网络, PFN) 框架，TimEE 只在合成 TSC 任务上进行 meta-training，每个任务包含由生成过程中的结构化分布偏移产生的、具有不同类别身份的时间序列。尽管预训练阶段没有见过真实时间序列，TimEE 在 UCR benchmark 上的 ROC AUC 排名第一、accuracy 排名第三，比较对象包括基础模型和有监督深度学习 baseline。据作者所知，TimEE 是第一个纯合成预训练、并在 UCR benchmark 上达到最先进性能的模型。这些结果表明，带合成先验的端到端 ICL 是 TSC 中一个很有吸引力但尚未充分探索的方向，扩展模型、先验设计和更丰富的生成机制都是自然的改进路径。代码已公开：http://github.com/automl/timee。
